## IA   

* Gemini 3.5
* Claude 4.5

    * **Claude 4.5** para scripting db

> **NOTA:** El uso de IA agilizo el desarrollo de la API (teniendo en cuenta la complejidad del proyecto), permitiendo ahorrar tiempo y recursos. Pudiendo crear la API en un tiempo menor al esperado.
